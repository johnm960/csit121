What is Javascript?

JavaScript is a scripting language of which users can use to create and control dynamic content within webpages.

JavaScript is related to HTML and CSS because it can be used to interact with the 2 aforementioned languages and follows a familiar logic flow into how they are programmed and can utilize methods that can alter the content of HTML and CSS pages.
